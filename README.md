# UART Transmitter (Verilog)

## Overview
This repository contains an implementation of a **UART Transmitter** written in Verilog. The goal of this project is to transmit 8-bit serial data from an FPGA to an external device (like a computer).

## Tools Used
* **Language:** Verilog HDL
* **IDE:** AMD Xilinx Vivado
* **Simulation:** Vivado Simulator
